# Zed Theme

This is a theme designed for a sleek and modern interface, inspired by the projects listed below.

---

## Credits

- **Nightfox UI Theme**: Based on the work from [cange/nightfox.zed](https://github.com/cange/nightfox.zed).
- **Andromeda Text Theme**: Utilizing styles and features from [EliverLara/Andromeda](https://github.com/EliverLara/Andromeda).
- **Andromeda Zed Port**: Adaptation by [ChocolateNao/andromeda-zed](https://github.com/ChocolateNao/andromeda-zed).

---

## Usage

### Manual Installation

1. Download the `.json` file with the theme variant of your choice from the [latest release](https://github.com/Fyelne/carbonfox-andromeda/releases/latest)
2. Navigate to `~/.config/zed/themes/` and place the file inside
3. Enter _theme selector: toggle_ in the command palette (<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>) and select __CarbonFox - Andromeda__ theme in the dropdown
